# eslint-config-vince

## Installation
```
npm install --save-dev @vince-cy-chao/eslint-config-vince
```

## Usage
```js
module.exports = {
  extends: ['@vince-cy-chao/eslint-config-vince'],
}
```

## Other configs
```js
module.exports = {
  extends: [
    '@vince-cy-chao/eslint-config-vince', 
    '@vince-cy-chao/eslint-config-vince/react'
  ],
}
```
